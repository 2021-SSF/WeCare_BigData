## WeCare
"WeCare"는 k-digital 융복합 프로젝트형 빅데이터 분석 서비스 과정(2021.06 ~ 2021.12, 6개월, 멀티캠퍼스)동안 학습한 모든 내용을 적용한 최종 프로젝트 입니다.
이 프로젝트는 프로젝트 기획, 개발, 시영상 제작, 발표까지 클라우드 / 인공지능 / IoT / 빅데이터의 각 분야의 팀원들과 협업한 대규모 협업 프로젝트 입니다.
프로젝트 기간 : 2021-10-28 ~ 2021-12-10
